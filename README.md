# Hangman-Game
A simple and interactive text-based Hangman Game built using Python. In this game, the player guesses a hidden word one letter at a time with a maximum of 6 incorrect attempts. The project is designed for beginners to practice core Python concepts like loops, conditionals, strings, lists, and the random module through a fun console-based game.


✨ Features

Uses 5 predefined words

Random word selection using Python’s random module

Tracks correct and incorrect guesses

Maximum of 6 wrong attempts

Simple and beginner-friendly console interface

No external libraries, graphics, or APIs required


🛠️ Concepts Used

random

while loops

if-else conditions

Strings

Lists

User input handling


🎯 Objective                                                                                                                                                                                                                         
Guess the hidden word before running out of attempts and win the game!
